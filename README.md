# Edu-Race 🚀

Edu-Race is a modern platform that brings academic competition between schools into a digital environment.

Students compete in areas such as Mathematics, Science and Software, while schools are ranked based on their performance.

---

## 🌐 Live Demo

https://edu-race.onrender.com

---

## Features

- School-based competition system  
- Live ranking dashboard  
- Multi-language support (TR / EN)  
- Modular backend structure  
- Supabase integration (database & auth)  

---

## Tech Stack

Backend:
- Python (Flask)

Frontend:
- HTML5, CSS3, Jinja2

Database:
- Supabase

## Installation

```bash
git clone https://github.com/Onurtuncer-6/edu-race.git
cd edu-race
```

```bash
python -m venv venv
```

```bash
pip install -r requirements.txt
```

---

## Environment Variables (.env)

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
```

---

## Run

```bash
python app.py
```
